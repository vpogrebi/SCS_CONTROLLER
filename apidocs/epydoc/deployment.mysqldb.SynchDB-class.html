connect(self, server, userid, passwd, socket, schema=None)

source code 

Connect to MySQL server

Parameters:
  • server - str - database server ('localhost' or <host>:<port>)
  • userid - str - user id on the <server>
  • passwd - str - user's password
  • socket - str or None - (optional) socket used to connect to database
  • schema - str - (optional) schema to be used for all database operations (default: None)

    This method connects to the database, sets <self.conn> attribute to Connection object, opens a cursor and sets <self.cursor> to that Cursor object. This attribute will be used for all subsequent database operations. At the same time, "autocommit" is set to True
