Module twisted_logger

source code

Created on Jul 29, 2009

Author: valeriypogrebitskiy

twisted_logger.py is intended to customize Twisted framework's logging functionality, as well as to simplify its use.

This module customizes logging format (prefix prepending actual log message) - which is set to: '%Y-%m-%d %H:%M:%S'. This format string get supplied to time.strftime() Python method:

prefix = time.strftime(self.timeFormat, time.localtime())

NOTE: ILogObeserver interface defined in this module really "belongs" to Twisted framework itself: it should have been defined within twisted.python.log module, but for some reason it is not there. Since I do not have permissions to write to twisted framework's files, I simply include this interface here

writeLog(logPrefix, logName, logMsg)

source code 

Write to a log file

Parameters:
  • logPrefix - 'system' prefix that will follow date/time prefix
  • logName - unique name (ID) identifying given log file
  • logMsg - main message to be written to a file

Note: logName log file identifier is used by observers (log.FileLogObserver instances) that "pick up" log messages designated for this particular log file from among all log messages produced by log.msg()

initLogging(name, logFile, logDir, parent=None)

source code 

Initialize log service that includes two files: LOG and ERROR file

Parameters:
  • name - unique log service name
  • logFile - base log file name shared between LOG and ERROR files
  • logDir - folder where LOG and ERROR files will be created (None can be provided to indicate that no files should be created)
  • parent - parent service (MultiService instance) to which this service should be added as subordinate service

Note: None should be provided as @param <logDir> to avoid creation of log files and service initialization alltogether
