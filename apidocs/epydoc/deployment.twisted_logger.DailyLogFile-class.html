Class DailyLogFile

source code

Customised version of twisted.python.logfile.DailyLogFile class. This class overrides suffix() method by providing custom implementation of that method.

Original (twisted) implementation of logfile.DailyLogFile.suffix() results in "<year>_<month>_<day>" being appended to log file name when it "rotates" into a new day. In addition, <month> and <day> portions of the suffix may be one character long (1 - 9), resulting in not very nice looking suffix like

2009_9_9

This custom implementation slightly improves file suffix to produce '%Y-%m-%d' format:

2009-09-09
