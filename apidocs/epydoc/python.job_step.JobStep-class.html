Class JobStep

source code

JobStep class - implements individual job step's functionality. Multiple (potentially) instances of this class belong to a single job. JobStep instances are initialized and orchestrated by a job.Job class.

Perform JobStep's finalization tasks. Completion of this method should trigger Job instance's callback() or errback() attached to <self.deferred>

Attention: This method should either throw RuntimeError exception, or return some (any) value indicating success. This should trigger Job instance's errback() or callback() method - if one was added to <self.deferred> by the Job instance before JobStep's run() is executed
