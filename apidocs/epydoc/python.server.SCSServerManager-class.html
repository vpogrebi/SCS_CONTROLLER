Class SCSServerManager

source code

SCSServer's main (public) class. Implements general SCS server management functionality.

Implemented using Singleton design pattern (extends <singleton.Singleton>) and ensures that class gets initialized only once - no matter how many times constructor is called (uses @singleton.uniq() decorator for its __init() constructor

Implements IResourceManager interface

Extends twisted.application.service.MultiService class. This (MultiService) functionality allows it to "wrap" (and expose to user) SCS servers as services that can be started and stopped alltogether when SCSServerManager service itself is being stopped
