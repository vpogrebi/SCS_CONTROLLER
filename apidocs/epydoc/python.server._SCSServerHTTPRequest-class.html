_SCSServerHTTPRequest class - responsible for processing HTTP requests that come from external clients

Inherits from twisted.web.http.Request. This class is responsible for following functionality related to client's job request management:

  1. Processing HTTP job request (starting Job instance that's responsible for executing a sequence of job steps that belong to workflow associated with this server)
  2. Responding to job 'STATUS' requests that can be submitted by external client
  3. Handling job completion event and reporting its result back to external client
