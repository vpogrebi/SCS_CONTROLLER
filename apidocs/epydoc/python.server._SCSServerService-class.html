Class _SCSServerService

source code

Extends twisted.application.service.MultiService class

This class "wraps" SCS server functionality (_SCSServerFactory/_SCSServerProtocol) as a service. It is responsible for starting/stopping the service (the server) and for initializing log and error files used by server entity.
