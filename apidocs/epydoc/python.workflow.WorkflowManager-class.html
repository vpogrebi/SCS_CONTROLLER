Class WorkflowManager

source code

Implements general SCS workflow management functionality. Implemented using Singleton design pattern (extends <singleton.Singleton>) and ensures that class gets initialized only once - no matter how many times constructor is called (uses @singleton.uniq() decorator for its __init() constructor

Implements IResourceManager interface
