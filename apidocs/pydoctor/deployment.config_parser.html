config_parser.py is a a "wrapper" for standard Python's ConfigParser.ConfigParser class.
Its main goal - is to simplify use of ConfigParser and provide a single-line way of 
loading configuration data from config. file

This functionality is implemented by Parser class, and simplified use of it
is exposed to users via single method getParams()

Examples:

        a. "config_parser.getParams(<path/file.cfg>)" - load config. parameters from ALL the sections in <path/file.cfg>
        b. "config_parser.getParams(<path/file.cfg>, 'SECTION1')" - load config. parameters from single section of <path/file.cfg>
        c. "config_parser.getParams(<path/file.cfg>, ('SECTION1', 'SECTION2')" - load config. parameters from multiple sections of <path/file.cfg>
