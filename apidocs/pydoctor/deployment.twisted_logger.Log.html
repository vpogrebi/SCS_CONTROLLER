Class d.t.Log(log.FileLogObserver):

Part of deployment.twisted_logger View In Hierarchy 

Customizes twisted.python.log.FileLogObserver class's implementation
by selecting to write to specific log file based on the value of 'logName'
entry in the <logEntryDict> dictionary that gets passed to emit() method

def emit(self, logEntryDict): 
Write to log file

@param logEntryDict: C{dict} containing key/value info about the message

@note: 'logName' entry of the @param <logEntryDict> is used to select messages
        designated to log file associated with this observer - from among all
        messages generated by log.msg()

@note: This method will "pick up" both LOG and ERROR messages that have same 
       logEntryDict['logName'] value. This is done intentionally - so that error
       messages get written both to LOG and ERROR file. To avoid this behavior,
       uncomment commented line that follows this document string
