Main purpose of mysqldb.py module - is to simplify MySQL database access and operations, as well as to customize some of the database operations.

This module provides three classes:

  1. SynchDB - Provides synchronous database operations using standard Python's MySQLdb module
  2. AsynchDB - Provides asynchronous database operations using standard Python's MySQLdb module
  3. TwistedMySQLdb - Implements asynchronous database operations using Twisted's 'adbapi' module ('ConnectionPool' class) and Python's MySQLdb module
