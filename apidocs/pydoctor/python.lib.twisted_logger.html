twisted_logger.py is intended to customize Twisted framework's logging functionality, as well as to simplify its use.

This module customizes logging format (prefix prepending actual log message) - which is set to: '%Y-%m-%d %H:%M:%S'. This format string get supplied to time.strftime() Python method:

prefix = time.strftime(self.timeFormat, time.localtime())

NOTE: ILogObeserver interface defined in this module really "belongs" to Twisted framework itself: it should have been defined within twisted.python.log module, but for some reason it is not there. Since I do not have permissions to write to twisted framework's files, I simply include this interface here
